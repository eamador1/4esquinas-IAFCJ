### Install

Install this project with:

VS Code, downloading repo.

### Usage

To run the project, download the repository and open it using your browser, right-click on the HTML file and select " Open with live server"

### Run tests

To run linter tests, run the following command:

npm install --save-dev hint@7.x
npx hint .
npm install --save-dev stylelint@13.x stylelint-scss@3.x stylelint-config-standard@21.x stylelint-csstree-validator@1.x
